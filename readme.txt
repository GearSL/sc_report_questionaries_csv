Задача которую решает данный скрипт:
Необходимо выгружать данные из SC:
1. Берем номер заявки из CSV файла
2. Берем старый статус из CSV файла
3. Формируем запросы для выборки следующих полей:
- номер займа (из CSV файла)
- статус прежний (из CSV таблицы)
- статус новый (из запроса, текущий статус заявки)
- кол-во дней просрочки 
- кол-во пролонгаций 
Номер заявки и прежний статус взять с прикрепленной таблицы
4. Формируем выходной CSV файл с выгруженными данными

p.s. скрипт можно выполнить через консоль "php index.php"
